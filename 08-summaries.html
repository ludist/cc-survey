8  Question Summaries

This Chapter consolidates the per-scenario results from Chapters 7.1–7.7 into cross-scenario summary tables for each question type. The tables replicate the Article’s summary tables programmatically from the underlying data; any discrepancies from the Article’s hardcoded values are documented in the Corrections section below.

Format conventions:

  • Q1–Q3 summary tables: Mean (SD) by group. Treatment groups additionally show [% difference from control]. Bold brackets indicate a significant pairwise difference from control (Dunn’s test, Holm-adjusted, p < .05).
  • Delta summary table: Mean (SD) by group. Brackets show the mean delta as a percentage of the group’s Q2 mean — the fraction of perceived consequences that exceeds respondent preferences. Bold brackets indicate a significant difference from control.
  • Q4 summary table: Median (SD) after 99th-percentile outlier trimming. Treatment groups show [% difference from control median]. Bold brackets indicate a significant difference from control.
  • VDA tables: Pairwise Vargha & Delaney’s A effect sizes with significance markers from Dunn’s post-hoc tests (Holm-adjusted). VDA = .50 indicates no group difference; VDA > .50 indicates the first-named group scored higher; VDA < .50 indicates the second-named group scored higher. Effect size thresholds follow Vargha & Delaney (2000): (S)mall \(\geq\) .56, (M)edium \(\geq\) .64, (L)arge \(\geq\) .71.
NoteEducational Use Scenario

Rows marked with a dagger (†) use supplemental Group B1 (n = 97) in place of Group B. See §3.1 for the B1 substitution.

```{r verify-consistency, include=FALSE}
# ── Verify scenario-level means match summary table computations ──
# This chunk runs silently. If a mismatch is found, it emits a warning
# visible in the render log.

scenarios <- list(
  list(code = "PD", data = survey_data),
  list(code = "PU", data = survey_data),
  list(code = "EU", data = survey_data_eu),
  list(code = "CA", data = survey_data),
  list(code = "LS", data = survey_data),
  list(code = "WN", data = survey_data),
  list(code = "FS", data = survey_data)
)

mismatches <- list()

for (sc_info in scenarios) {
  sc <- sc_info$code
  dat <- sc_info$data

  for (q_suffix in c("1", "2", "3", "25")) {
    col <- paste0(sc, q_suffix)
    if (!col %in% names(dat)) next

    fresh_means <- tapply(dat[[col]], dat$GROUP, mean, na.rm = TRUE)

    # Compare with summary table
    q_label <- switch(q_suffix, "1" = "q1", "2" = "q2", "3" = "q3", "25" = "delta")
    summary_tbl <- switch(q_label,
      q1 = summary_q1, q2 = summary_q2,
      q3 = summary_q3, delta = summary_delta
    )

    # The summary table cells contain formatted strings like "2.95 (2.06) [+15.3%]"
    # Extract the leading number for comparison
    summary_row <- summary_tbl[grepl(sc, summary_tbl$Scenario, ignore.case = TRUE), ]
    if (nrow(summary_row) == 0) next
    summary_row <- summary_row[1, , drop = FALSE]

    for (grp_col in c("Control (C)", "Short (A)")) {
      cell_val <- as.character(summary_row[[grp_col]])
      extracted <- as.numeric(sub("^([0-9.]+).*", "\\1", cell_val))
      grp_code <- ifelse(grepl("Control", grp_col), "C", "A")
      fresh_val <- unname(fresh_means[grp_code])

      if (length(extracted) == 1 && length(fresh_val) == 1 &&
          !is.na(extracted) && !is.na(fresh_val)) {
        if (abs(extracted - fresh_val) > 0.015) {
          msg <- sprintf("MISMATCH: %s %s Group %s -- summary: %.2f, fresh: %.2f",
                        sc, q_label, grp_code, extracted, fresh_val)
          mismatches <- c(mismatches, msg)
          warning(msg)
        }
      }
    }
  }
}

if (length(mismatches) == 0) {
  cat("Verification passed: all scenario means match summary table.\n")
} else {
  cat(sprintf("Verification found %d mismatches --- see warnings.\n", length(mismatches)))
}
```
