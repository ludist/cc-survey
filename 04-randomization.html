4  Randomization Check

4.1 Overview

Random assignment is the foundation of experimental causal inference. When participants are randomly assigned to treatment conditions, every pre-existing characteristic, observed or unknown, has an equal probability of appearing in each group. With sufficiently large samples, this probabilistic equivalence translates into actual balance: the groups become statistically indistinguishable on covariates that were not manipulated by the experimenter. The larger the sample, the more likely randomization equalizes groups across the full space of participant characteristics, including characteristics that were never measured. With n = 1,222 participants spread across three conditions (~400 per group), this section demonstrates that this experiment has sufficient power to achieve close balance on the five demographic covariates measured here.

Randomization checks verify that this equalization actually occurred and that group assignment was independent of pre-treatment participant characteristics. This Chapter reports covariate balance tests for Age, Income, Educational Attainment, Race, and Gender.

For each continuous or ordinal variable, we report a violin + box plot with group means and the corresponding omnibus test. Age and Income are tested with one-way ANOVA (parametric); Educational Attainment, an ordinal scale, is tested with the Kruskal-Wallis rank-sum test (non-parametric). For each categorical variable (Race, Gender), we report a contingency table and a Pearson chi-square test of independence.

Significance threshold: α = 0.05. A non-significant result (p > 0.05) means we fail to reject the null hypothesis that the groups are equivalent on that covariate.

While a null result is often disappointing in hypothesis testing, here it is the desired outcome: it indicates the experimental instrument is working as intended.

Note

Scope and sample. This Chapter tests covariate balance across Groups A, B, and C (n = 1,222). The B1 Educational Use supplement (n = 97) is excluded because B1 participants were recruited in a separate wave after the main data collection and did not complete the demographic battery; including them would confound the randomization check with between-wave differences. For details on the B1 supplement, see Codebook §2.1.3.

Randomization was performed by Qualtrics’s built-in survey flow randomizer, which assigned each participant to exactly one treatment condition at the start of the survey. The same randomization governed all seven scenarios — participants saw the same condition (C, A, or B) throughout. The scenarios themselves were also presented in a random order to prevent order effects. For the full experimental design, see Chapter 1 (Methods).

Note

Relationship to population analysis. This chapter assesses internal validity, or whether the three treatment groups are comparable to each other. Chapter 5 (Sample v. Population) assesses external validity, or how the combined sample compares to the U.S. adult population via contemporaneous American Community Survey benchmarks. Both analyses examine the same five demographic variables but answer different questions.

4.1.1 How to Read These Graphs

The continuous-variable sections below display violin + box plots styled to match the scenario analyses in Chapter 7. Each plot layers several visual elements:

  • Violin (the outer curved shape): Shows the probability density of responses at different values. Wider sections indicate more observations clustered at that value.
  • Box plot (the inner rectangle): The bottom edge marks Q1 (25th percentile), the top edge marks Q3 (75th percentile), and the horizontal line inside is the median. Whiskers extend to 1.5× the interquartile range beyond Q1 and Q3.
  • Mean point and label (the purple dot with a number): Shows the group mean for each treatment condition.
  • Caption: Reports the omnibus test results, including the test statistic, degrees of freedom, p-value, and sample size. ANOVA is used for age and income; the Kruskal-Wallis rank-sum test is used for educational attainment (ordinal).

Quick interpretation: If the p-value in the caption exceeds 0.05, the groups are balanced on that variable. The collapsible interpretation box below each section confirms the result in plain language.

4.7 Summary

Note

Summary interpretation. 5 of 5 covariates show balanced distributions (p > .05) across the three treatment groups (all marked “n.s.” in the Sig. column).

All five demographic covariates (age, household income, educational attainment, race, and gender) pass the balance test at α = .05. Three continuous/ordinal variables were tested with omnibus comparison tests (ANOVA for age and income; Kruskal-Wallis for educational attainment), and two categorical variables were tested with Pearson chi-square tests of independence (race and gender). None reached statistical significance, confirming that Qualtrics’s survey-flow randomizer successfully distributed participants across conditions without systematic demographic confounds.

The balance documented here supports a causal interpretation of between-group differences in the scenario analyses (Chapter 7): because the groups were demographically equivalent before treatment exposure, observed differences in infringement perceptions or licensing valuations can be attributed to the experimental manipulation (license notice format) rather than to pre-existing demographic differences between groups.

This Chapter confirms internal validity, or comparability between treatment groups. For external validity, or how the pooled sample compares to the U.S. adult population. See Chapter 5 (Sample v. Population Analysis).

# Helpers for consistent p-value and statistic extraction across test types.
# Used in both table construction and inline R interpretation text.

rand_p <- function(result) {
  if (result$test_type == "parametric") {
    summary(result$test)[[1]][["Pr(>F)"]][1]
  } else {
    result$test$p.value
  }
}

rand_stat <- function(result) {
  if (result$test_type == "parametric") {
    summary(result$test)[[1]][["F value"]][1]
  } else {
    as.numeric(result$test$statistic)
  }
}

rand_df <- function(result) {
  if (result$test_type == "parametric") {
    summary(result$test)[[1]][["Df"]][1]
  } else {
    result$test$parameter
  }
}

rand_test_label <- function(result) {
  if (result$test_type == "parametric") "ANOVA" else "Kruskal-Wallis"
}

# ── Custom randomization plot (matches 7.X scenario style) ──
# Violin + boxplot + mean dots with omnibus test in caption.
# No VDA brackets (we expect non-significance).
create_rand_plot <- function(data, dv, group_var = "GROUP",
                             title = NULL, test_type = "parametric",
                             base_family = PLOT_FONT) {
  # Reorder groups: C, A, B
  group_label_map <- c("C" = "Control (C)", "A" = "Short (A)",
                        "B" = "Full + Primer (B)")
  data[[group_var]] <- factor(data[[group_var]],
                              levels = c("C", "A", "B"),
                              labels = group_label_map[c("C", "A", "B")])

  # Group means
  group_stats <- data %>%
    dplyr::group_by(!!rlang::sym(group_var)) %>%
    dplyr::summarise(mean_val = mean(!!rlang::sym(dv), na.rm = TRUE),
                     .groups = "drop")

  # Omnibus test caption
  formula_obj <- as.formula(paste(dv, "~", group_var))
  if (test_type == "parametric") {
    test_result <- aov(formula_obj, data = data)
    s <- summary(test_result)
    f_val <- s[[1]][["F value"]][1]
    p_val <- s[[1]][["Pr(>F)"]][1]
    df1 <- s[[1]][["Df"]][1]; df2 <- s[[1]][["Df"]][2]
    n_total <- nrow(data)
    p_text <- if (p_val < 0.001) "< .001" else sprintf("= %.3f", p_val)
    caption_md <- sprintf(
      "ANOVA *F*(%d, %d) = %.2f, *p* %s, *n* = %d",
      df1, df2, f_val, p_text, n_total)
  } else {
    kw <- kruskal.test(formula_obj, data = data)
    kw_eff <- rstatix::kruskal_effsize(data, formula_obj)
    n_total <- sum(!is.na(data[[dv]]))
    p_text <- if (kw$p.value < 0.001) "< .001" else sprintf("= %.3f", kw$p.value)
    caption_md <- sprintf(
      "Kruskal-Wallis *H*(%d) = %.2f, *p* %s, *e*<sup>2</sup> = %.2f, *n* = %d",
      kw$parameter, kw$statistic, p_text, kw_eff$effsize, n_total)
  }

  # Build plot
  p <- ggplot2::ggplot(data, ggplot2::aes(
    x = !!rlang::sym(group_var), y = !!rlang::sym(dv)
  )) +
    ggplot2::geom_boxplot(width = 0.15, outlier.shape = NA, fill = "#F0F0F0") +
    ggplot2::geom_violin(width = 1.4, fill = VIOLIN_FILL, color = VIOLIN_OUTLINE,
                         alpha = 0.4, linewidth = 0.5) +
    ggplot2::geom_point(data = group_stats,
      ggplot2::aes(x = !!rlang::sym(group_var), y = mean_val),
      size = 5, color = MEAN_DOT_COLOR, inherit.aes = FALSE) +
    ggplot2::geom_label(data = group_stats,
      ggplot2::aes(x = !!rlang::sym(group_var), y = mean_val,
                   label = sprintf("%.2f", mean_val)),
      size = 5.5, family = base_family, nudge_x = 0.20,
      label.size = 0.15, label.padding = ggplot2::unit(0.12, "lines"),
      fill = "white", inherit.aes = FALSE) +
    ggthemes::theme_tufte(base_family = base_family, base_size = 18) +
    ggplot2::theme(
      plot.title          = ggtext::element_markdown(size = 22,
                              margin = ggplot2::margin(b = 2)),
      plot.title.position = "plot",
      plot.caption        = ggtext::element_markdown(size = 17, hjust = 0.5),
      axis.text.x         = ggplot2::element_text(size = 18, color = "black",
                              margin = ggplot2::margin(t = -10)),
      axis.text.y         = ggplot2::element_text(size = 16, color = "black",
                              margin = ggplot2::margin(r = 0)),
      axis.title.y        = ggplot2::element_blank(),
      axis.ticks.y        = ggplot2::element_blank(),
      axis.ticks.x        = ggplot2::element_blank(),
      plot.background     = ggplot2::element_rect(fill = "transparent", colour = NA),
      panel.background    = ggplot2::element_rect(fill = "transparent", colour = NA),
      plot.margin         = ggplot2::margin(t = 5, r = 0, b = 5, l = 10)
    ) +
    ggplot2::scale_x_discrete(expand = ggplot2::expansion(add = c(0.0, 0.0))) +
    ggplot2::coord_cartesian(clip = "off") +
    ggplot2::labs(title = title, x = NULL, caption = caption_md)

  p
}

decode_race <- function(x) {
  x <- as.character(x)

  ifelse(is.na(decoded), x, decoded)

## Summary {#sec-rand-summary}

```{r}
#| label: tbl-rand-summary
#| tbl-cap: "Randomization Check Summary — Covariate Balance Across Treatment Groups (A, B, C; n = 1,222)"

rand_summary <- summarize_randomization_checks(rand_results)

# Reformat for display
rand_display <- data.frame(
  Variable = paste0("`", rand_summary$Variable, "`"),
  Type     = rand_summary$Type,
  Test     = rand_summary$Test,
  `P Value`  = sprintf("%.4f", rand_summary$P_Value),
  Sig.     = ifelse(rand_summary$Significant, "\\*", "n.s."),
  Status   = rand_summary$Status,
  check.names = FALSE,
  stringsAsFactors = FALSE
)
knitr::kable(rand_display, align = c("l", "l", "l", "r", "c", "l"))
```

::: {.callout-note collapse="true"}
**Summary interpretation.** `r sprintf("%d of %d", sum(!rand_summary$Significant), nrow(rand_summary))` covariates show balanced distributions (*p* > .05) across the three treatment groups (all marked "n.s." in the Sig. column).

All five demographic covariates (age, household income, educational attainment, race, and gender) pass the balance test at *α* = .05. Three continuous/ordinal variables were tested with omnibus comparison tests (ANOVA for age and income; Kruskal-Wallis for educational attainment), and two categorical variables were tested with Pearson chi-square tests of independence (race and gender). None reached statistical significance, confirming that Qualtrics's survey-flow randomizer successfully distributed participants across conditions without systematic demographic confounds.

The balance documented here supports a causal interpretation of between-group differences in the scenario analyses ([Chapter 7](07-scenarios.qmd)): because the groups were demographically equivalent before treatment exposure, observed differences in infringement perceptions or licensing valuations can be attributed to the experimental manipulation (license notice format) rather than to pre-existing demographic differences between groups.

This Chapter confirms **internal validity**, or comparability between treatment groups. For **external validity**, or how the pooled sample compares to the U.S. adult population. *See* [Chapter 5 (Sample v. Population Analysis)](05-population.qmd).
:::
