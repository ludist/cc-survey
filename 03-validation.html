## Delta Variables {#sec-delta-validation}

Delta variables (`Δ` = `Q2` − `Q3`) are created during data preparation by `create_all_deltas()`. They represent the gap between what respondents believe the law says (`Q2`) and what they believe it should say (`Q3`).

| `Δ` Value | Interpretation |
|---------|----------------|
| Positive | Law perceived as more strict than preferred |
| Zero | Perceived law matches preferred rule |
| Negative | Law perceived as more lenient than preferred |

```{r delta-validation}
delta_vars <- paste0(c("PD", "PU", "EU", "CA", "LS", "WN", "FS"), "25")
delta_present <- delta_vars[delta_vars %in% names(survey_data)]

cat("Delta variables present: ", paste(delta_present, collapse = ", "), "\n")
cat("Count: ", length(delta_present), "of 7\n")

# Quick sanity check: verify delta = Q2 - Q3
for (sc in c("PD", "PU", "CA", "LS", "WN", "FS")) {
  q2  <- paste0(sc, "2")
  q3  <- paste0(sc, "3")
  d25 <- paste0(sc, "25")
  if (all(c(q2, q3, d25) %in% names(survey_data))) {
    computed <- survey_data[[q2]] - survey_data[[q3]]
    match <- all(computed == survey_data[[d25]], na.rm = TRUE)
    cat(sprintf("  %s: Q2 - Q3 == Delta? %s\n", sc, ifelse(match, "✓", "✗ MISMATCH")))
  }
}
```
